31 Badges can be used as part of links or buttons to provide a counter.
32 
33 {% example html %}
34 <button type="button" class="btn btn-primary">
35  Notifications <span class="badge badge-light">4</span>
36 </button>
37 {% endexample %}
38 
39 Note that depending on how they are used, badges may be confusing for users of screen readers and similar assistive technologies. While the styling of badges provides a visual cue as to their purpose, these users will simply be presented with the content of the badge. Depending on the specific situation, these badges may seem like random additional words or numbers at the end of a sentence, link, or button.
40 
41 Unless the context is clear (as with the "Notifications" example, where it is understood that the "4" is the number of notifications), consider including additional context with a visually hidden piece of additional text.
42 
43 {% example html %}
44 <button type="button" class="btn btn-primary">
45  Profile <span class="badge badge-light">9</span>
46  <span class="sr-only">unread messages</span>
47 </button>
48 {% endexample %}

62 ## Pill badges
63 
64 Use the `.badge-pill` modifier class to make badges more rounded (with a larger `border-radius` and additional horizontal `padding`). Useful if you miss the badges from v3.
65 
66 {% example html %}
67 {% for color in site.data.theme-colors %}
68 <span class="badge badge-pill badge-{{ color.name }}">{{ color.name | capitalize }}</span>{% endfor %}
69 {% endexample %}

71 ## Links
72 
73 Using the contextual `.badge-*` classes on an `<a>` element quickly provide _actionable_ badges with hover and focus states.
74 
75 {% example html %}
76 {% for color in site.data.theme-colors %}
77 <a href="#" class="badge badge-{{ color.name }}">{{ color.name | capitalize }}</a>{% endfor %}
78 {% endexample %}
