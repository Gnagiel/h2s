9 ## Examples
10 
11 Alerts are available for any length of text, as well as an optional dismiss button. For proper styling, use one of the eight **required** contextual classes (e.g., `.alert-success`). For inline dismissal, use the [alerts jQuery plugin](#dismissing).
12 
13 {% example html %}
14 {% for color in site.data.theme-colors %}
15 <div class="alert alert-{{ color.name }}" role="alert">
16  This is a {{ color.name }} alert—check it out!
17 </div>{% endfor %}
18 {% endexample %}

23 ### Link color
24 
25 Use the `.alert-link` utility class to quickly provide matching colored links within any alert.

44 </div>

48 ### Dismissing
49 
50 Using the alert JavaScript plugin, it's possible to dismiss any alert inline. Here's how:
51 
52 - Be sure you've loaded the alert plugin, or the compiled Bootstrap JavaScript.
53 - If you're building our JavaScript from source, it [requires `util.js`]({{ site.baseurl }}/docs/{{ site.docs_version }}/getting-started/javascript/#util). The compiled version includes this.
54 - Add a dismiss button and the `.alert-dismissible` class, which adds extra padding to the right of the alert and positions the `.close` button.
55 - On the dismiss button, add the `data-dismiss="alert"` attribute, which triggers the JavaScript functionality. Be sure to use the `<button>` element with it for proper behavior across all devices.
56 - To animate alerts when dismissing them, be sure to add the `.fade` and `.show` classes.
57 
58 You can see this in action with a live demo:
59 
60 {% example html %}
61 <div class="alert alert-warning alert-dismissible fade show" role="alert">
62  <strong>Holy guacamole!</strong> You should check in on some of those fields below.
63  <button type="button" class="close" data-dismiss="alert" aria-label="Close">
64  <span aria-hidden="true">&times;</span>
65  </button>
66 </div>
67 {% endexample %}

69 ## JavaScript behavior
70 
71 ### Triggers
72 
73 Enable dismissal of an alert via JavaScript:
74 
75 {% highlight js %}
76 $('.alert').alert()
77 {% endhighlight %}
78 
79 Or with `data` attributes on a button **within the alert**, as demonstrated above:
80 
81 {% highlight html %}
82 <button type="button" class="close" data-dismiss="alert" aria-label="Close">
83  <span aria-hidden="true">&times;</span>
84 </button>
85 {% endhighlight %}
86 
87 Note that closing an alert will remove it from the DOM.
88 
89 ### Methods
90 
91 | Method | Description |
92 | --- | --- |
93 | `$().alert()` | Makes an alert listen for click events on descendant elements which have the `data-dismiss="alert"` attribute. (Not necessary when using the data-api's auto-initialization.) |
94 | `$().alert('close')` | Closes an alert by removing it from the DOM. If the `.fade` and `.show` classes are present on the element, the alert will fade out before it is removed. |
95 | `$().alert('dispose')` | Destroys an element's alert. |
96 
97 {% highlight js %}$(".alert").alert('close'){% endhighlight %}

99 ### Events
100 
101 Bootstrap's alert plugin exposes a few events for hooking into alert functionality.
102 
103 | Event | Description |
104 | --- | --- |
105 | `close.bs.alert` | This event fires immediately when the <code>close</code> instance method is called. |
106 | `closed.bs.alert` | This event is fired when the alert has been closed (will wait for CSS transitions to complete). |
107 
108 {% highlight js %}
109 $('#myAlert').on('closed.bs.alert', function () {
110  // do something…
111 })
112 {% endhighlight %}
