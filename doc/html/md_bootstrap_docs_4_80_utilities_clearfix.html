clearfix

layout: docs title: Clearfix description: Quickly and easily clear floated content within a container by adding a clearfix utility. group: utilities

toc: true

Easily clear floats by adding .clearfix to the parent element. Can also be used as a mixin.

{% highlight html %}

...

{% endhighlight %}

{% highlight scss %} // Mixin itself clearfix() { &::after { display: block; content: ""; clear: both; } }

// Usage as a mixin .element {

; } {% endhighlight %}

The following example shows how the clearfix can be used. Without the clearfix the wrapping div would not span around the buttons which would cause a broken layout.

{% example html %}

<button type="button" class="btn btn-secondary float-left">Example Button floated left</button> <button type="button" class="btn btn-secondary float-right">Example Button floated right</button>

{% endexample %}
